# Air Residences Authorization Letter

A browser-based authorization-letter generator prepared for deployment with GitHub Pages.

## Publish with GitHub Pages

1. Push this folder to a GitHub repository.
2. In the repository, open **Settings → Pages**.
3. Under **Build and deployment**, choose **Deploy from a branch**.
4. Select the `main` branch and `/ (root)`, then save.

GitHub will publish `index.html` as the site homepage.

## Run locally

Open `index.html` directly in a browser, or serve this directory with any static web server.

## Note

The JPEG export loads `html2canvas` from cdnjs, so that feature needs an internet connection. The form, preview, signature images, and browser print/PDF flow are otherwise contained in `index.html`.
